## Objective
The objective of this task is to build an executive-ready Business Intelligence dashboard that shows key performance indicators (KPIs), trends, and insights using a retail sales dataset.

## Tools Used
- Power BI Desktop (Primary Tool)
- Dataset: Global Superstore

## KPIs Created
- Total Sales
- Total Profit
- Profit Margin

## Dashboard Components
- KPI Cards (Sales, Profit, Margin)
- Sales Trend Chart (Sales over Time)
- Category-wise Sales Breakdown (Bar Chart)
- Region-wise Performance (Map/Bar Chart)
- Top 10 Products Table
- Slicers for Region, Date, and Category

## Insights
1. Technology category generated the highest sales.
2. West region produced the highest profit.
3. Sales were highest in the fourth quarter.
